Project Description:
This project aims to scrape real estate data from MercadoLibre (a popular online marketplace) to analyze apartment prices in Barquisimeto, Venezuela. The goal is to assess the market, identify the most affordable apartments, and analyze pricing trends based on location, size, and amenities. By utilizing web scraping techniques in Python with the Scrapy library, the project collects relevant data such as apartment details, price, and other features. The data is then cleaned and processed for further analysis, allowing us to compare square meter prices and identify sellers offering competitive deals.

Key Features:
Data Scraping: Using Python and Scrapy to collect apartment listings from MercadoLibre.
Data Cleaning: Processing data in PostgreSQL for easy manipulation and analysis.
Price Analysis: Calculating square meter prices and comparing them to identify the best deals.
Seller Analysis: Investigating which sellers have the most properties and offering the best prices.
Interactive Exploration: Filtering and sorting apartments based on various criteria, such as number of rooms, bathrooms, and price.

Technologies Used:
Python: Scrapy, Pandas, NumPy
SQL: PostgreSQL
Web scraping: Scrapy
Data Analysis: Pandas, NumPy


Steps:
Data Scraping: Collect apartment listings from MercadoLibre.
Data Cleaning: Transform and filter data for consistency and ease of analysis.
Price Calculation: Calculate price per square meter and identify affordable apartments.
Seller Analysis: Group apartments by seller to analyze their inventory and average prices.
Data Export: Export the processed data to CSV for further evaluation.
Results:
The analysis identifies apartments with lower price per square meter compared to the market average.
It helps users find the most competitive sellers and apartment deals in the target location.

Descripción del Proyecto:

Este proyecto tiene como objetivo extraer datos de bienes raíces de MercadoLibre (una plataforma popular de comercio en línea) para analizar los precios de apartamentos en Barquisimeto, Venezuela. La meta es evaluar el mercado, identificar los apartamentos más asequibles y analizar las tendencias de precios según la ubicación, tamaño y características. Utilizando técnicas de web scraping en Python con la librería Scrapy, el proyecto recopila datos relevantes como los detalles de los apartamentos, precio y otras características. Luego, los datos son limpiados y procesados para su análisis, lo que permite comparar los precios por metro cuadrado e identificar los vendedores que ofrecen las mejores ofertas.


Características Clave:

Web Scraping: Uso de Python y Scrapy para recopilar anuncios de apartamentos en MercadoLibre.
Limpieza de Datos: Procesamiento de datos en PostgreSQL para su manipulación y análisis.
Análisis de Precios: Cálculo del precio por metro cuadrado y comparación para identificar las mejores ofertas.
Análisis de Vendedores: Investigación sobre qué vendedores tienen más propiedades y ofrecen los mejores precios.
Exploración Interactiva: Filtrado y ordenación de apartamentos según varios criterios, como número de habitaciones, baños y precio.

Tecnologías Utilizadas:

Python: Scrapy, Pandas, NumPy
SQL: PostgreSQL
Web Scraping: Scrapy
Análisis de Datos: Pandas, NumPy


Pasos:

Web Scraping: Recopilar anuncios de apartamentos de MercadoLibre.
Limpieza de Datos: Transformar y filtrar los datos para su consistencia y facilidad de análisis.
Cálculo de Precios: Calcular el precio por metro cuadrado e identificar los apartamentos más asequibles.
Análisis de Vendedores: Agrupar apartamentos por vendedor para analizar su inventario y precios promedio.
Exportación de Datos: Exportar los datos procesados a CSV para su evaluación posterior.
Resultados:
El análisis identifica apartamentos con precios por metro cuadrado más bajos en comparación con el promedio del mercado.
Ayuda a los usuarios a encontrar a los vendedores más competitivos y las mejores ofertas de apartamentos en la ubicación objetivo.
